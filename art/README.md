# Screenshots

Drop your captures here with these exact filenames — the main README references them:

| File | What to capture |
|---|---|
| `editor-crop.png`   | The editor in **Crop** mode (preset row + crop frame) |
| `editor-adjust.png` | The editor in **Adjust** mode (brightness/contrast/saturation rulers) |
| `editor-filter.png` | The editor in **Filter** mode (filter strip) |

Optional:

| File | What to capture |
|---|---|
| `demo.gif` | A short screen recording of a full crop → done flow |

**How to capture**
- **iOS Simulator:** `File → New Screen Recording`, or `⌘S` for a screenshot.
- **Android Emulator:** the camera/record icons in the emulator toolbar.
- To turn a recording into a GIF: `ffmpeg -i demo.mov -vf "fps=15,scale=320:-1" demo.gif`

Keep images reasonably small (portrait phone screenshots ~1080px wide is plenty).
